**[GitHub](https://github.com/Mainaky/Multi-Agent-Healthcare-AI-System)** • LangChain, FastAPI, RAG, Google Gemini

• Implemented keyword-based RAG pipeline grounded on 15 clinical guidelines (ADA, AHA, KDIGO, ACC/AHA) for evidence-based decisions on FHIR-compatible data.  
• Built real-time evaluation framework for routing confidence, latency & task completion.

• Engineered U-Net CNN achieving 97.2% segmentation accuracy; VGG19 transfer learning reached 99.1% classification accuracy (outperforming ANN baseline 82.4% and standard CNN 94.8%).  
• Integrated Grad-CAM heatmaps for clinical explainability; deployed via automated CI/CD pipeline (GitHub → Jenkins → Docker → DockerHub).
